*********This is the instruction onthe operation of the application**************
The github link for the application has 3 branches 
1. Backend - Built with PHP (Laravel)
2. Frontend - Built with Vuejs
3. How-to - Instruction for the operation of the application.


**********Backend************
-- Run *git clone -b backend https://github.com/azeemade/Teesco.git*
-- Go to the folder application using cd command on your cmd or terminal
-- Run *composer install* on your cmd or terminal
-- Copy .env.example file to .env on the root folder. You can type copy .env.example .env if using command prompt Windows or cp .env.example .env if using terminal, Ubuntu
-- Open your .env file and change the database name (DB_DATABASE) to whatever you have, username (DB_USERNAME) and password (DB_PASSWORD) field correspond to your configuration.
-- Run *php artisan key:generate*
-- Run *php artisan migrate*
-- Run *php artisan serve*
-- Go to http://localhost:8000/

*************Frontend*******************
-- Run *git clone -b frontend https://github.com/azeemade/Teesco.git*
-- Go to the folder application using cd command on your cmd or terminal
-- Install nodejs
-- Run *npm install*
-- Run *npm run dev*
